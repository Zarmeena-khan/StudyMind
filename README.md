# StudyMind

A premium AI-powered Notes Simplifier web app built with React, Vite, and Express.

## Features
- Text paste, PDF upload, and handwritten image notes input
- Summary, flashcards, and quiz generation
- Gemini API-powered knowledge extraction
- Luxury dark academic styling with animated interactions

## Setup
1. Copy `.env.example` to `.env.local`
2. Add your Gemini API key to `.env.local`
3. Install dependencies: `npm install`
4. Run development mode: `npm run dev`

## Notes
- The frontend runs at `http://localhost:5173`
- The backend API listens on `http://localhost:4000`
- The Vite dev server proxies `/api` to the backend
